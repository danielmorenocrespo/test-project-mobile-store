# ProjectMobileStore
Instructions for preparing and launching the project





--------Dependencies to Install---------

Node.js 18+ (includes npm)

Angular CLI 19+




--------Environment --------

1. Install Node.js

Download and install from:
https://nodejs.org

Verify the installation:

node -v
npm -v




2. Install Angular CLI

Install globally:

npm install -g @angular/cli


--------Verify the installation--------

ng version







--------Available Scripts--------

Start:
npm run start
Enter the following route in your browser: http://localhost:4200/

Build:
npm run build

Tests:
npm run test

Lint:
npm run lint
